## Featured Projects

### 🇸🇪 Skattråd
Agentic AI compliance assistant for Swedish SMEs using **Google ADK** and **NVIDIA NIM**, providing grounded responses from official government sources.

### 🔥 Temperature Field Reconstruction
Machine learning framework for reconstructing combustion temperature profiles from laser absorption spectroscopy using **Multi-Output Gaussian Processes**.

### 🧠 HybridMorph
Published research introducing a hybrid training framework for medical image registration by combining synthetic and real MRI data.

**Publication:** *HybridMorph: Bridging the Gap between Synthetic and Real Data for Accurate MR Image Registration* (NCVPRIPG 2025)

### 📡 Complex-Valued Neural Networks
Deep learning methods for Polarimetric SAR image classification using complex-valued neural networks.

### 📖 MAML & FOMAML
Implementation and comparison of gradient-based meta-learning algorithms for few-shot learning using PyTorch.
